SimplexGridFactory

Provides SimplexGridBuilder, a convenience interface to rhe triangle (via Triangulate.jl) and TetGen (via TetGen.jl) mesh generators. Code using it needs to pass the Triangulate resp. the TetGen module as parameter to the constructor of SimplexGridBuilder and thus needs to adhere to their respective license conditions. This package itself is MIT licensed.
